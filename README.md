# Appointment System

## 📋 About the Project

An Appointment Management System built with **Java**, allowing users to book and manage appointments through a graphical user interface (GUI). The project uses Maven for dependency management and includes unit tests with JUnit and Mockito to ensure code quality.

## 🛠️ Technologies Used


- **Maven** — Project and dependency management
- **JUnit** — Unit testing
- **Mockito** — Object mocking in tests
- **GUI (Swing)** — Graphical user interface
